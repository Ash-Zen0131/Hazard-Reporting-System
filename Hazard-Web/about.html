Known Hazard Reporting System

The Known Hazard Reporting System is a crowdsourcing application that allows users to report and view location-based hazards.

The application supports three categories: road hazards, environmental hazards, and building hazards.

The system consists of an Android mobile application, Cloud Firestore database, and a web dashboard hosted through Firebase Hosting.
